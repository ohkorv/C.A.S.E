# Project Details
C.A.S.E is a custom macropad made to fit my needs. It has 8 Baby Kangaroo Cherry MX Switches, a rotary encoder with a button usb c port and is powered by a esp32-s3.


# Repo  Structure
- `Libs/` Contains 3d models, symbols and footprints for Kicad
- `Blender/` It has the files neccesary for blender rendering.
- `Production/pcb` PCB fabrication files (Gerbers, BOM, Pick & Place)
- `Production/Case` F3d and step files of the custom case I made in Fusion 360
- `Renders/` Rendered pictures of the final product
- `Hardware/` Pcb and schematic from Kicad
- `Software/Companion App`  Companion app for controlling C.A.S.E
- `Software/MacroPad software`  Code to be put on the C.A.S.E
